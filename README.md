## Overview

Sentinel is an end-to-end content moderation system that uses a fine-tuned **DistilBERT** transformer model to detect toxic language in real-time. It ships with a premium glassmorphism web UI, a FastAPI backend, and achieves strong accuracy on the Jigsaw Toxic Comment dataset.

## Features

- 🧠 **DistilBERT Transformer** — Fine-tuned on 50k samples from the Jigsaw Toxic Comment dataset
- ⚡ **Real-time Analysis** — Sub-second inference with toxicity probability scoring
- 🎨 **Premium UI** — Glassmorphism design with animated arc gauge, particle background, and scan history
- 🔌 **REST API** — Clean FastAPI backend with automatic Swagger documentation
- 📊 **Detailed Results** — Toxicity percentage, confidence score, and 5-tier verdict classification
- 🗂️ **Scan History** — LocalStorage-backed history of past analyses
- ⌨️ **Keyboard Shortcuts** — `Ctrl + Enter` for quick analysis
- 🧪 **Quick Test Chips** — One-click example inputs (Friendly, Hostile, Neutral, Abusive)

## Quick Start

### 1. Install Dependencies

```bash
pip install -r requirements.txt
```

### 2. Train the Model

```bash
python train_transformer.py
```

Fine-tunes DistilBERT on the Jigsaw dataset (~50k samples). Training takes approximately 15–30 minutes on CPU with frozen base layers.

### 3. Launch the App

```bash
uvicorn app:app --reload
```

Visit **http://localhost:8000** to use the web interface.

## API Reference

### `POST /predict`

Analyze text for toxicity.

**Request:**
```json
{
  "text": "Your text here"
}
```

**Response:**
```json
{
  "toxicity": 0.9234,
  "label": "toxic"
}
```

### `GET /health`

Check API and model status.

```json
{
  "status": "healthy",
  "model_loaded": true,
  "model_dir": "sentinel_model"
}
```

### `GET /docs`

Interactive Swagger API documentation (auto-generated by FastAPI).
